## 四、排查记录（关键）

测试过程中曾出现全部 `code:1001 Invalid Token`，排查确认：

1. **直连上游验证**：用同一 token 直接调用 `sider.ai` 返回 `code:0`（额度 remain:99999），证明 **token 有效**（sider.ai 未严格校验 JWT exp，尽管 JWT 标称 2026-01-20 过期）。
2. **根因**：8000 端口存在一个**未加载 token 的旧 Deno 实例**。清理全部 deno 进程后，用 `--env-file=.env` 干净重启，启动日志确认 `SIDER_AUTH_TOKEN ✅ 已配置`，问题解决。
3. **结论**：代理请求管线（路由→上下文构建→上游转发→错误处理）工作完全正常。
